Local Regression and GAMs

Nov 28 2019 Interviews 2 minutes read (About 334 words)

Local Regression

What?

It is an approach for fitting flexible non-linear functions, which involves computing the fit at a target point using only the nearby training observations.

Details

  • Gather the fraction $s=k/n$ of training points whose predictors values are close to that of the target points.
  • Assign a weight to each point in the neighborhood, so that the point furthest from the target point has weight zero, and the closest has the highest weight.
  • Fit a weighest least squares regression for a prediction for the target point.

Pros and Cons

  • (+) such varying coefficient models are a useful way of adapting a model the most recently gathered data.
  • (+) We can also fit the model using several chosen variables instead of all variables.
  • (-) in high dimensions, when p is much larger than 3 or 4, local regression can perform poorly because there will generaly be very few training obversations close to the target point.

Generalized Additive Models (GAM)

What?

Generalized Additive Models provide a general framework for extending a standard linear model by allowing non-linear functions of each of the variables, while maintaining additivity.

Details

$y_i=\beta_0+f_1(x_{i1})+f_2(x_{i2})+…f_p(x_{ip})+\epsilon_i$

$log(\frac{p(x)}{1-p(x)})=\beta_0+\beta_1f(X_1)+…\beta_pf_p( X_p)$

It is called an additive model because we calculate a separate $f_i$ for each $X_j$, and then add together all of their contributions.

Pros and Cons

  • (+) allows us to fit a non-linear $f_j$ to each $X_j$ so that we can automatically model non-linear relationships that standard linear regression will miss. This can lead to more accurate predictions.
  • (+) We can still examine the effect of each $X_j$ on $Y$ individually while holding all of the other variables fixed.
  • (-) model is restricted to be additive. With many variables, important interactions can be missed. For fully general models, we have to look for even more flexible approaches such as random forest and boosting.
