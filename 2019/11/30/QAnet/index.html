QAnet

Nov 30 2019 Projects 7 minutes read (About 996 words)

RNNs with attention –> convolution and self-attention

Motivation

RNNs with attention are often slow for both training and inference due to the sequential nature of RNNS.

So QANet: encoder consists exclusively of convolution and self-attention, where convolution models local interactions (captures the local structure of text) and self-attention models global interactions. (between each pair of words) Hence we can speed up and train more data.

Comparison

The main difference is that QANet user convolutions and self-attentions as the building blocks of encoders that separately encodes the query and context.

Advantage

  • efficient reading comprehension model using convolution and self-attention as building encoder blocks
  • higher accuracy with novel data augmentation to enrich the trianing data by paraphrasing.

Detail

context paragraph: $C={c_1,c_2,…c_n}$ and query (question) sentence: $Q={q_1,q_2,…,q_m}$. Output a span $S={c_i,c_{i+1},…,c_{i+j}}$

High-level

five major components:

  • embedding layer (fixed word embedding + character embedding)
  • embedding encoder layer (convolution+self-attention)
  • Context-query attention layer
  • model encoder layer (convolution+self-attention)
  • output layer

Convolution

convolution is a mathematial operation on two functions $f$ and $g$ that produces a third function expressing how the shape of one is modified by the other. it is defined as the intergral of the product of the two functions after one is reversed and shifted.

  • $(f*g)(t)=\int_{\infty}^{\infty}f(\gamma)g(t-\gamma)d\gamma$

Here $t$ needs not representation the time domain. The convolution formula can be described as a weighted average of the function $f(\gamma)$ at the moment $\gamma$ where the weighting is given by $g(-\gamma)$ simply shifted by amount $t$. as $t$ changes, the weighting function emphasizes different parts of the input function.

Convolutional Neural Networks

Convolutional neural network is a specialized type of neural network model designed for working with two-dimensional image data.

Central to the convolutional neural network is the convolutional layer that gives the network its name. This layer performs an operation called a “convolution“.

convolution is a linear operation that involves the multiplication of a set of weights with the input, much like a traditional neural network. the multiplication is performed between an array of input data and a two-dimensional array of weights, called a filter or a kernel: input #dot product# filter=single value.

using a filter smaller than the input is intentional as it allows the same filter (set of weights) to be multiplied by the input array multiple times at different points on the input.

using a filter (filter size) smaller than the input (input length) is intentional as it allows the same filter to be multiplied by the input array multiple times at different points on the input. Specifically, the filter is applied systematically to each overlapping part or filter-sized patch of the input data, left to right, top to bottom.

The output from multiplying the filter with the input array one time is a single value. as the filter is applied multiple times to the input array, the result is two-dimensional array output values that represent a filtering of the input. As such, the two-dimensional output array from this operation is called feature space

channels:

color images have multiple channels, typically one for each color channel, such as green, red and blue. That is, a single image provided as input to the model is three images.
